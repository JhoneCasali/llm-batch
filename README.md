## 📋 What is llm-batch?

llm-batch is a lightweight C++ library designed to improve how you handle requests to large language model (LLM) APIs. It groups requests together to reduce delay and lower costs. This means faster responses and less money spent when working with these APIs.

You do not need to install extra software or dependencies. The library comes as a single file, which makes it easy to include and use. llm-batch helps developers and services that use LLMs work more effectively.

## 🔧 How Does llm-batch Work?

llm-batch groups several API requests into batches before sending them. This reduces the delay caused by sending individual requests, and lowers costs if the API charges per request.

The library works fine for many types of LLM APIs. It handles the timing and grouping so programmers don't have to do it themselves.
